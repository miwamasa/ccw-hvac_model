# ビルエネルギーシミュレーションアプリ - クイックスタート

## 🚀 すぐに始める

### 1. インストール（初回のみ）

```bash
# 依存パッケージのインストール
pip install -r requirements.txt
```

### 2. アプリケーション起動

```bash
# Streamlitアプリの起動
streamlit run app.py
```

ブラウザが自動的に開きます（通常 http://localhost:8501）

### 3. 基本的な使い方（3ステップ）

#### ステップ1: プリセット選択
- サイドバーで「最新オフィス」または「旧式オフィス」を選択

#### ステップ2: シミュレーション実行
- 「シミュレーション」タブで「🚀 シミュレーション実行」ボタンをクリック

#### ステップ3: 結果確認
- 「結果」タブでグラフを確認
- 「データ」タブでCSVダウンロード

## 📊 サンプル実行

コマンドラインから実行する場合：

```bash
# サンプルスクリプトの実行
python sample_run.py
```

以下のファイルが生成されます：
- CSVファイル（シミュレーション結果）
- JSONファイル（設定）
- PNGファイル（比較グラフ）

## 🧪 テスト実行

```bash
# テストの実行
python test_building_energy_model.py
```

すべてのテストが成功すれば正常に動作しています。

## 💡 便利な機能

### 比較モード
- サイドバーで「比較モード（最新 vs 旧式）」をチェック
- 2つのケースを同時に比較できます

### 設定の保存・読み込み
- サイドバーの「💾 保存」「📁 読み込み」ボタンを使用
- 設定をJSONファイルとして保存・共有できます

### カスタム設定
- プリセット選択で「カスタム設定」を選択
- すべてのパラメータを自由に変更できます

## 🆘 トラブルシューティング

### アプリが起動しない場合

```bash
# Pythonバージョン確認（3.8以上が必要）
python --version

# 依存パッケージの再インストール
pip install -r requirements.txt --upgrade
```

### ポート競合の場合

```bash
# 別のポートで起動
streamlit run app.py --server.port 8502
```

## 🎯 主な機能

### 計算機能
- ✅ 月別熱負荷計算（顕熱・潜熱）
- ✅ 全館空調システムのエネルギー計算
- ✅ 個別空調システムのエネルギー計算
- ✅ 空気線図の状態点計算

### 可視化機能
- ✅ システム別エネルギー消費量比較グラフ
- ✅ 負荷内訳グラフ（熱量ベース）
- ✅ 月別推移グラフ
- ✅ 比較モード（2ケース同時表示）

### データ管理
- ✅ CSV形式での結果保存
- ✅ JSON形式での設定保存・読み込み
- ✅ プリセット機能（最新/旧式オフィス）
